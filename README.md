# Genetic-Algorithms-for-Sort-Based-Clustering-py-

This repository makes use of a few datasets from the web and accompanies a non-scientific paper on the matter if you're interested.

To run simply add your .data file into a folder inside the /data folder as in "/data/yourDataset/dataset.data"
This will automatically be read by the code, IMPORTANT: remember that these files are supposed to be uniform data-files with commas separating values.
